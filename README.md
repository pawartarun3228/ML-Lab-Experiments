# ML Lab Experiments

A collection of Machine Learning lab experiments implemented in Python using Jupyter Notebooks. This repository covers key ML concepts including unsupervised clustering and supervised learning through hands-on datasets.

---

## Experiments

### 1. Mall Customers Segmentation (`mallcustomers.ipynb`)
**Technique:** Unsupervised Learning — K-Means Clustering

Performs customer segmentation on the Mall Customers dataset. Customers are grouped based on features like age, annual income, and spending score to help businesses understand customer behavior and target groups effectively.

**Key Concepts:**
- Exploratory Data Analysis (EDA)
- K-Means Clustering
- Elbow Method for optimal cluster selection
- Cluster visualization

---

### 2. Student Performance Analysis (`studentperformance.ipynb`)
**Technique:** Supervised Learning — Regression / Classification

Analyzes and predicts student academic performance based on features such as study hours, parental education, and test preparation. Helps identify factors that significantly impact student outcomes.

**Key Concepts:**
- Data Preprocessing & Feature Engineering
- Regression / Classification Models
- Model Evaluation (Accuracy, RMSE, etc.)
- Data Visualization

## Getting Started

1. **Clone the repository**
```bash
   git clone https://github.com/pawartarun3228/ML-Lab-Experiments.git
   cd ML-Lab-Experiments
```

2. **Install dependencies**
```bash
   pip install numpy pandas matplotlib seaborn scikit-learn jupyter
```

3. **Launch Jupyter Notebook**
```bash
   jupyter notebook
```

4. Open any `.ipynb` file and run the cells.
